# LedCube.Streamer App

## Features

- Connect to a LED Cube via Newtork
  - Support Broadcast Discovery in the Local Network
- Playback Controls
  - Support Play, Pause, Stop, Scrub through Animations
  - Speed Up/Down Animations
- Load Animations from a File
- Support Dynamic Animations
  - Support Animation configurations (for Plugin Animations)
  - Save Dynamic Animation configurations to a File
- Support for Playlists
  - Features to curate a Playlist of animations
  - Change/Update their order, add new ones, remove them
  - Store a Playlist as a File
  - Change Timing
  - Repeat Settings (For Animations or complete Playlist)
